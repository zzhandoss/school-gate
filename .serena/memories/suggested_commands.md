# Suggested Commands (PowerShell)

## Project
- Install: `pnpm i`
- Dev: `pnpm dev`
- Lint: `pnpm lint`
- Lint fix: `pnpm lint:fix`
- Typecheck: `pnpm typecheck`
- Test: `pnpm test` (also `pnpm test:local`)
- Build: `pnpm build`
- DB generate: `pnpm db:generate`
- DB migrate: `pnpm db:migrate`

## Workspace Utilities
- List files: `Get-ChildItem`
- Read file: `Get-Content`
- Search (if available): `rg "pattern"`
